1979年生まれ。東京都出身。
2014年まではIT関連ではない仕事に就いていましたが、
将来について考える上で、自分にとって「ものづくり」が大切だという気持ちが大きくなり、
退職して卒業訓練校でJava/Androidアプリ開発を学習しました。
ここで、Google Play Storeでのリリースまでを経験。
2015年から、AWSを利用したデータ分析基盤の構築や保守作業を担当しました。
ITの基本を中心に、Java / Pythonなどの理解を深めましたが、
ものづくりに近づくために2017年に退職。
卒業訓練校のWebクリエーター科でHTML/CSS/Photoshopなどを学習しながら、
独学でも、JavaScript / Sass / CSS設計（OOCSS、BEM）などについて学習しました。
現在、コーディングとデザインをどちらもできるWEBクリエーターを目指しています。
